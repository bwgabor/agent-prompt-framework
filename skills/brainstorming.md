# Purpose

Facilitate a focused brainstorming session that generates and evaluates viable ideas before committing to a direction.

# When to Use

Use when the user needs to explore multiple options before selecting an idea, approach, or starting execution.

# Inputs

- A topic, problem, or opportunity to explore
- Any relevant goals, constraints, or evaluation criteria

# Process

1. Clarify the objective and any important constraints.
2. Generate multiple distinct ideas without filtering.
3. Group similar ideas and remove clear duplicates.
4. Evaluate the remaining ideas against the stated criteria.
5. Present viable options with a short rationale for each.

# Expected Output

A prioritized set of viable ideas, each with a short rationale and enough detail to select or develop further.
